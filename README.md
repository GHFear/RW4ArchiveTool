# RW4ArchiveTool
*RW4 Archive parser, unpacker, decompressor and packer.* <br>

**Supports:** <br>
-XSF (Compression Type 1, 2, 3) <br>
-PSF (Compression Type 1, 2, 3) <br>
-WSF (Compression Type 1, 2, 3) <br>
-BIG ("EB" Archive) (No Decompression Yet) <br>
-BIG ("BIG4" Archive) <br>
-BIG ("BIGF" Archive) <br>

**Working on:** <br>

-RX2 (Arena File) <br>
